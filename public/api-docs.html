📋 API接口

获取随机图片

GET /api/random
参数（可选）：
category - 图片分类（landscape, anime, beauty, nature, city, art）
orientation - 图片方向（landscape=横屏/PC, portrait=竖屏/手机）
返回：直接返回图片文件

获取随机图片信息

GET /api/random/json
参数：同上
返回：JSON格式的图片信息

获取图片列表

GET /api/images
page - 页码（默认1）
limit - 每页数量（默认12）
category - 图片分类（可选）
orientation - 图片方向（可选）

获取指定图片

GET /api/images/:id
返回：指定ID的图片文件

📱 设备适配

横屏图片 (landscape)

适合PC端、网页背景、桌面壁纸等场景

https://rp.itdianbao.com/api/random?orientation=landscape

竖屏图片 (portrait)

适合手机端、移动应用、竖屏显示等场景

https://rp.itdianbao.com/api/random?orientation=portrait

💡 使用示例

获取风景类横屏图片

GET /api/random?category=landscape&orientation=landscape

获取动漫类手机壁纸

GET /api/random?category=anime&orientation=portrait

获取图片信息（JSON）

GET /api/random/json?category=nature

JavaScript调用示例

// 设置随机背景图 document.body.style.backgroundImage = 'url(https://rp.itdianbao.com/api/random?category=landscape)'; // 获取图片信息 fetch('https://rp.itdianbao.com/api/random/json?category=anime') .then(response => response.json()) .then(data => console.log(data));
